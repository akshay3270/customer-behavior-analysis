# Customer Behavior Analysis Dashboard

## Overview
This project focuses on analyzing customer purchasing behavior using an interactive Power BI dashboard. The goal is to extract meaningful insights that help businesses understand customer trends and improve decision-making.

## Objectives
- Analyze customer purchase patterns
- Identify high-value customers
- Track key performance indicators (KPIs)

## Features
- Interactive dashboard with filters and slicers
- Revenue and sales trend analysis
- Customer segmentation insights
- KPI metrics such as total revenue and average order value

## Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- Excel (Dataset)

## Key Insights
- Identified top-performing customer segments
- Observed trends in customer purchasing behavior
- Provided insights for improving customer retention

## Project Files
- Power BI Dashboard (.pbix)
- Dashboard Screenshots

## Conclusion
This project demonstrates how data visualization and analysis can be used to understand customer behavior and support data-driven business decisions.
